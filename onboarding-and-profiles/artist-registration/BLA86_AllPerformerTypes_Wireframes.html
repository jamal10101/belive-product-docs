What's unique about each type:

🎸 Band (3 screens): Band name + year formed → Add Members (min 3, each with role: Lead Vocalist, Guitarist, Bassist, etc.) → Members shown in review. Each member can be found on BeLive or added manually. Admin (creator) gets 👑 icon.

👥 Duo (1 screen): Act name → Exactly 2 members required. Simpler than Band — just add 1 partner with their role. Partner receives invite.

💃 Troupe (1 screen): Troupe/group name → Number of members (count, not individual listing) → Performance style picker (Classical, Bollywood, Contemporary, etc.) → Only key members listed (optional). Theatre/dance groups don't need every performer listed.

🎧 DJ Act (1 screen): DJ name → Setup type (Solo, DJ+MC, DJ+Live, B2B) → Primary genre (EDM, Bollywood, House, etc.) → Equipment (Own CDJs, Own Controller, Need Venue Setup). If DJ+MC, can add MC partner.

✨ Other (1 screen): Free-text type input with smart suggestions (Mime, Puppeteer, Fire Dancer, etc.) → Art form description → Performance setup (Solo/Partner/Group). Enables BeLive to discover new performer types.

After Step 1b, ALL types flow into the same shared steps: Step 2 (Categories) → Step 3 (Details: DOB, photo, city, contact, socials) → Step 4 (Review) → Step 5 (Pending Verification)
