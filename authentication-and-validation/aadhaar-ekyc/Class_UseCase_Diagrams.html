Relationships:
• ArtistRegistration → uses → AadhaarKYCService (for verification)
• AadhaarKYCService → produces → eKYCData (verification result)
• ArtistRegistration → uses → ContentGateService (for age-based routing)
• AdminKYCQueue → reads → eKYCData (to review verification)
• All classes → use → AuditLogger (for compliance trail)
• ContentGateService → reads → eKYCData.ageTier (to determine content access)
